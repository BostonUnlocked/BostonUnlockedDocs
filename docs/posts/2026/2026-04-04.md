### Infrastructure Improvement and New Server Status Page

As we deepen our collaboration with the MADE, we've been focused on backend changes to support deploying, monitoring and maintaining BostonUnlocked more sustainably on the MADE's Azure infrastructure. As a result, the repository now has automatic CI builds, an automated patching system was implemented (triggered by a maintainer command) and the server also has a monitoring service in addition to the core backend service. The monitoring service has led to a cool new feature: we have a live server status page!

![Server Status Page](/img/posts/2026-04-04_server-status-page.jpg)

This is currently a basic version which refreshes at most once every 10 seconds and is served via HTTP only (so HTTPS warnings when accessing it are normal). To view it, simply open the community server's IP in a browser (how to find it is mentioned in [Quick Start](/quick_start)).

Non-critical bugfixes were temporarily paused during this important enhancement.
