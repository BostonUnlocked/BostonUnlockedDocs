#### BostonUnlocked - The game is back

As many of you have noticed in other threads, the game has been revived thanks to the extraordinary efforts of ShadowAid and with the collaboration of the Museum of Art and Digital Entertainment (MADE) via ZiggyTheHamster (who allowed us to use the decompilation of the original server binaries).

The revived free and open source server project is called BostonUnlocked. Check the website[bostonunlocked.github.io] to get started and join our Community Discord Server[discord.gg] for help and finding players.

Thanks to the many testers and contributors we already have on our Discord Server, the revived server has been confirmed to be very stable and the main game content (including DLC) is fully playable. It's still being developed and we could use help in finding and fixing bugs though so do join in!

The server allows self-hosting to play single player offline, connecting to another server (we have an open alpha server) and hosting your own multiplayer server (if you know what you're doing). All details are on the website linked above.

Play however you want to play and get back in the game Chummers!
