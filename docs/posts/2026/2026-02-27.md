## 2026-02-27 Open Alpha Server

Here's another big announcement: We now have an open alpha server you can simply connect to to play the game (Steam version only for now)!

1. Download the patching program from:

<https://raw.githubusercontent.com/BostonUnlocked/BostonUnlockedServer/refs/heads/main/clientsetup/patch_embedded_configs.exe>

2. Run it by double clicking on it
3. It will open one file browser asking you to point to the resources.assets file in the Steam installation folder for the game.
4. After this, in the command line window after it will ask for a host, simply enter this one: 40.124.88.244
5. Once done, simply launch the game normally and enjoy!

This is an open alpha so expect bugs, the server may not always be available and save data may get deleted. Please report your bugs in the ⁠support channel for now and remember the limitations set out in the readme here:

<https://github.com/BostonUnlocked/BostonUnlockedServer>

Have fun!
