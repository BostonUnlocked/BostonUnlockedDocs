Previously, our [Game Reference](/gameref/) section specified drop tables as unlisted as they vary per player profile. The [Loot Drop Tables Reference](/gameref/loot_drop_tables_reference) is now live on the documentation site.

### Respec

A long-requested feature is now available: you can respec your character via the `/resetskills` [Global Chat Command](/commands_player) (which is no longer restricted to [Admin Global Chat Commands](/commands_admin)). This undoes skill selections and refunds karma without losing story progress, allowing you to modify your character build. Every character gets 1 free respec but the next respec is only available after 28 days on the community server (self-hosting grants admins unlimited respecs).
