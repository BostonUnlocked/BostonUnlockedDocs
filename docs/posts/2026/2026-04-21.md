### Respec

A long-requested feature is now available: you can respec your character via the `/resetskills` [Global Chat Command](/commands_player). This command is no longer restricted to [Admin Global Chat Commands](/commands_admin)). Respeccing undoes skill selections and refunds karma without losing story progression, allowing you to modify your character's build. Every character gets 1 free respec but the next respec is only available after 28 days on the community server (self-hosting grants admins unlimited respecs).

There were indications in the code that the developers intended to add a respec feature and several BostonUnlocked players requested it as well. This deviation from full vanilla behaviour is therefore in keeping with what we can make out of the original direction of the game along with discussions with the community so far. This is an experimental feature so let us know what you think about it on our Discord server! Whether we keep this feature or remove it from general availability is entirely up to you!
