# Enabling DLC

1. Launch the game
2. From the game's launcher, click on Coupons
3. Add the coupon codes in this document to enable the DLC you want.

::: warning List Contents
This list includes all DLC, even previously restricted backer rewards so everyone can experience all the content officially released
:::

::: info

1. This coupon list is based off the [corresponding document](https://github.com/BostonUnlocked/BostonUnlockedServer/blob/main/docs/coupon_codes.md) on the server repository.
2. The coupon code list was simplified from the original as there was a lot of overlap (the current version still unlocks everything, you just do so with less codes to enter). If you wish to refer to the original full coupon code list, see [this older version of the file](https://github.com/BostonUnlocked/BostonUnlockedServer/blob/31bc6fc2a6ce7cb2683df4e49cdb75a95213b87a/docs/coupon_codes.md).
:::

## Consolidated List of All Coupon Codes

This section lists all coupon codes currently accepted by the server's coupon system and the rewards they grant.

::: info Unlocks Persist Account-Wide
Package coupon rewards are applied to all existing occupied careers and future occupied careers for the account.
:::

Copy and claim these one by one in the launcher to activate the corresponding DLC. The contents for each code are listed in subsequent sections of this page.

```
    SRO-Addon1
    SRO-Addon2
    SRO-GAME-KEY
    SRO-ITEM-CATALYST
    SRO-ITEM-CATALYST-SKUA
    SRO-ITEM-CATALYST-SPRAYDOWN
    SRO-ITEM-COMMUNITY-GOODIES1
    SRO-ITEM-GAMESROCKET
    SRO-ITEM-HAREBRAINED
    SRO-ITEM-LAGISSUES1
    SRO-ITEM-PAX-18
    SRO-ITEM-PEGASUS
    SRO-ITEM-STARTER-GEAR
    SRO-KS-AWAKENED
    SRO-KS-COOP
    SRO-KS-MASTER
    SRO-ReflexRecorder
    SRO-STEAM-DLC-DELUXE
```

::: warning Subsequent Sections only Detail DLC Contents
Once you have entered all the codes above, you do not need to enter any more codes to unlock all DLC. The sections after this one on this page only detail the contents of each coupon code.
:::
