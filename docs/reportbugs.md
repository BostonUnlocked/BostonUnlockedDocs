# Reporting Bugs

Reporting bugs you encounter is vital to reviving the game as it lets us know what needs fixing on the server.

To report a bug, simply do this:

1. When you encounter a bug, as quickly as you can, type this in the game chat:

```
/bug <short description of the bug>
```

Where \<short description of the bug\> is, as the text suggests, a short description of the bug you encountered. This helps us locate relevant server-side logs if you are on the community server. If running locally or self-hosting multiplayer, run this anyway and send us your logs attached to a Github issue as described below.

2. Log a detailed issue on the code repository by [clicking on this link](https://github.com/BostonUnlocked/BostonUnlockedServer/issues/new/choose) and then choosing "Bug report"

3. Fill in the detailed bug report by answering the questions in the template. This is very important for bug-fixing as the details help identify root causes more than a short description on its own might.
