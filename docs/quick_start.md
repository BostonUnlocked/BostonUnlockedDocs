# Quick Start Guide

We have an open alpha community server you can easily connect to to play the game!

Here's how:

1. Join our [Community Discord](https://discord.gg/cyuPZrC8Wr)
2. Go to the `#how-to-connect` channel to get the community server IP to use
3. Follow the instructions in [Connect to Another Server](/connectmp) to patch the game to use that IP and have fun!

::: info
We do not mention the server IP here to protect it from automated scanners on GitHub which might misuse the information
:::
