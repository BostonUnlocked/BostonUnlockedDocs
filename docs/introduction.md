# BostonUnlocked

BostonUnlocked is a fully community-driven Shadowrun Chronicles: Boston Lockdown Server Reimplementation. It aims to provide an experience which as close as possible to the original (aka vanilla) final version of Shadowrun Chronicles: Boston Lockdown game server when it was last officially online (client version 23.3.28073.265425).

This free server has been reverse engineered from the official game client as well as the original server binaries (the latter with the permission and collaboration of our friends at the [Museum of Art and Digital Entertainment (MADE)](https://www.themade.org/)). A clean-room reverse engineering approach was used. The reimplemented server contains no proprietary code or assets and requires official game client files to work.

The [BostonUnlockedServer GitHub repository](https://github.com/BostonUnlocked/BostonUnlockedServer) contains tooling and a C# local service that lets the Shadowrun Chronicles: Boston Lockdown client talk to the reimplemented server (BostonUnlocked).

In short, you can run your own server to play the game and optionally set it up to invite friends to play with you.
