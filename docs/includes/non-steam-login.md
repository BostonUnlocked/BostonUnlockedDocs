## Setting up without Steam

::: info Non-Steam players only
The following step is only required if you do **not** own the game on Steam.
:::

1. Download [`LauncherConfig.xml`](https://raw.githubusercontent.com/BostonUnlocked/BostonUnlockedServer/refs/heads/main/clientsetup/LauncherConfig.xml) from the official repository.
2. Place `LauncherConfig.xml` in the same directory as `shadowrun.exe`.

::: info
`shadowrun.exe` is located in the root of your game installation folder:

`<GameRoot>\shadowrun.exe`

where `<GameRoot>` refers to where you installed the game.
:::

## Non-Steam login

If you are attempting to login without having a Steam instance running, enter an email address and password to create an account or login.

::: danger No password recovery
Passwords **cannot** be reset and email addresses are **not** verified. If you forget your password, you will be **permanently locked out of your account**. Please record your login credentials carefully!
:::
