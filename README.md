# BostonUnlockedDocs

Documentation and website for the BostonUnlockedServer project hosted at:

<https://github.com/BostonUnlocked/BostonUnlockedServer>

## Running the site locally

1. You will need to install [NodeJS](https://nodejs.org/) to build and run the website locally:

2. After installing NodeJS, clone the documentation source code:

```sh
git clone https://github.com/BostonUnlocked/BostonUnlockedDocs
cd BostonUnlockedDocs
```

3. Run the following command to start the website:

```sh
npm run docs:dev
```

4. The site will now be accessible from http://127.0.0.1:5173/ (or any other port shown on the terminal)

**Note:** This is the repository for the BostonUnlocked website code and content. For the server source code, go to [BostonUnlockedServer](https://github.com/BostonUnlocked/BostonUnlockedServer)
